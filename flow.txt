1. Project Setup
Clone the repo (if not local):

Create a virtual environment:

bash
python -m venv venv  
Activate it:

Linux/Mac: source venv/bin/activate

Windows: venv\Scripts\activate

Install dependencies:

bash
pip install -r requirements.txt  
Set up .env

bash
mkdir -p data/{resumes,job_descriptions,embeddings}  
Run initial setup:

bash
python wsgi.py  
Creates admin user: admin@alpha.ae / Alphadata.123.

3. Run the Application
Start Flask:

bash
flask run  
Access: http://localhost:5000
